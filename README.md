# Get-Local-Info

> Web application with the purporse of showing weather and timezone information of any selected city in the world

## Technologies Used

- ReactJS
- CSS(Flexbox, Grid)
- Context API
- Open Weather Map API
- TzLookup API

## Project Status

Project is: _in progress_
